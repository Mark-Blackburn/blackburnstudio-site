# Blackburn Studio Documentation

## Purpose

This documentation set records the current website, captures provisional strategic direction, and provides a structure for future technical and business decisions.

Current-state documents describe the implemented site as it exists in the repository.

Strategy documents may contain recommendations and working assumptions that are not yet approved.

## Document status definitions

- `Current state`: describes the implemented repository or deployed behaviour as currently verified.
- `Provisional`: working recommendation or framing that still requires owner review.
- `Proposed`: recommendation advanced for decision but not yet approved.
- `Approved`: confirmed direction or decision.
- `Superseded`: retained for record, but no longer current.
