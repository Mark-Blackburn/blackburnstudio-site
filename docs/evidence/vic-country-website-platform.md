## Engagement origin

The broader engagement began with a request to:

- migrate Google-hosted email to Microsoft 365
- assist with rolling out the supporting infrastructure

The website-platform opportunity emerged later.

The initial website-related request was to create simple SharePoint pages for individual clubs.

Mark Blackburn identified that:

- the pages were intended to be externally accessible
- SharePoint was not appropriate as the public website layer
- individually maintained club pages would not provide a scalable or consistent solution

Mark Blackburn then conceived a shared website platform that generates and supports club websites using centrally managed SharePoint data.

Do not treat the full website platform as part of the original request.
